**Status:** Complete

---

## 1. Objective

Evaluate Big5Loop's zero-shot LLM-based OCEAN personality detector against the PANDORA dataset (real Reddit posts with self-report BFI ground truth) and characterize the performance ceiling for single-post, zero-shot inference.

This complements the Phase 4 PERSONAGE evaluation (r=0.457 mean Pearson) by testing on ecologically valid, unconstrained human text rather than machine-generated restaurant reviews.

---

## 2. Dataset

**Source:** PANDORA Talks: Personality and Demographics on Reddit (Gjurković et al., ACL 2020)  
**HuggingFace:** `jingjietan/pandora-big5`

**Structure:** Each row = 1 Reddit author with 1 representative text sample and self-report BFI scores (percentiles 0–100, mapped to [-1, 1] via `(x - 50) / 50`).

**Text statistics (test set):**
- Median length: 103 characters
- Mean length: 219 characters
- 75th percentile: 239 characters
- 30% of samples are under 50 characters (e.g., "lol nice", "agreed")

## 3. Key Difference: PERSONAGE vs. PANDORA

| Factor | PERSONAGE (Phase 4) | PANDORA (Phase 5) |
|--------|--------------------|--------------------|
| Text source | Machine-generated by PERSONAGE NLG system (Mairesse & Walker, 2007) | Real Reddit posts by human authors |
| Ground truth source | Human raters judging perceived personality of the text | Self-report BFI questionnaire (author's self-perception) |
| GT–text alignment | **Direct** — text was engineered to express specific personality levels | **Indirect** — writing style ≠ self-reported personality |
| Domain | Restaurant reviews only (constrained) | Any Reddit topic (unconstrained) |
| Text length | ~230 chars, consistent | 1–9,922 chars, highly variable |
| Personality signal | Strong, systematic (encoded by generation parameters) | Weak, incidental (buried under topic, context, mood) |
| Achieved mean Pearson r | **0.457** | **0.07–0.12** |

The PERSONAGE NLG system parameterizes over 60 linguistic features (hedging, verbosity, formality, lexical choice, etc.) to encode personality directly into text surface form. When human raters judge these texts, they perceive the intended personality — creating high GT–text alignment. The same Llama 3.3 70B model with similar prompting detects this signal reliably (r=0.457).

On PANDORA, the GT comes from authors' self-report questionnaires — a fundamentally different construct from observable writing style. A person scoring high-N (self-reported anxiety) may write calm, composed posts. A person writing aggressively online may score high-A on the BFI. This construct mismatch, not the detector, is the primary bottleneck.

## 4. Methods

### 4.1 Label Normalization

Raw PANDORA Big Five columns are percentile scores on a 0–100 scale. The preprocessing script maps them to [-1, 1]:

```
norm = clip((x - 50) / 50, -1, 1)
```

Implemented in `scripts/preprocess_pandora.py`. This aligns the numeric range with Big5Loop's [-1, 1] detector outputs.

### 4.2 Evaluation Regimes

Two complementary evaluation paths were used:

1. **End-to-end** (`run_pandora_eval.py`): Calls Big5Loop's `/api/chat` endpoint with `evaluation_mode: true`. Measures system-level agreement including gateway behavior, timeouts, and version drift.

2. **Offline detection** (`llama31_five_trait_detect.py`): Direct LLM API calls bypassing the application stack. Allows systematic variation of prompt, temperature, few-shot count, and architecture.

### 4.3 Prompt & Model Harness

The harness (`benchmark_llama31_prompt_tuning.py`) performs structured grid search over model, prompt set, and few-shot count. For each configuration it computes per-trait Pearson r, Spearman ρ, MAE, bias, bootstrap 95% CIs (B=1000), and a composite ranking score:

```
composite = 0.5 × macro_r + 0.3 × coverage + 0.2 × (1 - macro_MAE)
```

### 4.4 Detector Configuration (Offline Experiments)

- **Model:** `meta/llama-3.3-70b-instruct`
- **API:** JuLing relay (`open.177911.com/v1/chat/completions`)
- **Architecture:** All-at-once (single API call returns all 5 traits as JSON)
- **Prompt style:** `strict_style` — "You are a psychometric rater. Infer Big Five O,C,E,A,N from writing style only. Ignore topic popularity, morality, and factual correctness. Return only JSON with O,C,E,A,N in [-1,1]."
- **Post-hoc linear calibration:** Per-trait `gt ≈ a·pred + b` fitted on evaluation set

## 5. Key Findings

### 5.1 The Zero-Shot Ceiling

All prompt engineering strategies converge to **macro Pearson r ≈ 0.07–0.12** on PANDORA regardless of configuration. This represents the zero-shot LLM ceiling for single-post personality detection against self-report ground truth.

Strategies systematically tested and found to produce equivalent results:
- **Few-shot count:** 0, 3, 10, 20 shots → no significant difference
- **Temperature:** 0.05 vs 0.3 → higher temperature improves score variance but not correlation
- **Prompt complexity:** Minimal instruction vs. rubric-based vs. contrastive vs. chain-of-thought debiased → all equivalent
- **Architecture:** All-5-at-once vs. per-trait separate calls → equivalent (per-trait costs 5× API calls)
- **Prompt ensemble:** Averaging 8 different prompt strategies → no improvement over single best prompt
- **Text length filter:** ≥200 chars → does not reliably improve on larger sample

### 5.2 Score Variance Problem

At low temperature (t=0.05), the model produces extremely narrow prediction ranges (pred_std ≈ 0.03–0.10 vs. gt_std ≈ 0.52–0.62). Predictions cluster around 2–3 discrete values per trait. Raising temperature to 0.3 resolves this, producing pred_std ≈ 0.46–0.64 matching GT variance.

### 5.3 Per-Trait Signal Strength

Across experiments, the strongest detectable traits are:
- **Neuroticism** (r ≈ 0.08–0.16) — emotional volatility in writing style
- **Agreeableness** (r ≈ 0.08–0.13) — interpersonal warmth/hostility cues

The weakest:
- **Extraversion** (r ≈ -0.11 to +0.04) — frequently inverted; the model equates online verbosity with extraversion, but many prolific online writers are introverts

### 5.4 Text Length Effect (Within-Experiment)

Stratifying Experiment C (100 mixed-length samples) by text length:

| Text length | Macro r | n |
|-------------|---------|---|
| < 50 chars | -0.008 | 30 |
| ≥ 50 chars | 0.141 | 70 |
| ≥ 100 chars | 0.163 | 52 |
| ≥ 300 chars | 0.187 | 24 |

Posts under 50 characters carry zero personality signal. However, this within-experiment finding did not generalize to Experiment F (200 fresh long-text samples), where macro r was 0.072 — suggesting the within-experiment result was inflated by calibration overfitting on small subgroups.

### 5.5 Few-Shot Ordering Effect (Harness Finding)

Early harness experiments revealed that few-shot example ordering matters:

- **Workflow order (first 5):** Macro Pearson = 0.0903
- **Balanced high/low round-robin:** Macro Pearson ≈ 0.1700 (+88%)

The balanced strategy selects examples in round-robin order across `(O_high → O_low → C_high → C_low → ...)`, ensuring each trait receives both positive and negative exemplars. However, in the subsequent systematic experiments (§4.7), this advantage did not persist — all few-shot strategies converged to r ≈ 0.10 at larger sample sizes, suggesting the initial finding was amplified by small-n variance.

### 5.6 Multi-Agent Architecture (Invalidated)

A 5-specialized-agent architecture (`multi_agent_ocean.py`) was tested, with each Big Five trait handled by a dedicated agent with a focused prompt. A dry-run simulation (n=12) reported macro Pearson r=0.95 — however, this was subsequently **invalidated** by real API experiments (Experiment D in §4.7), which showed the per-trait approach achieves r ≈ 0.11 at n=100, equivalent to the all-at-once approach at 5× API cost. The dry-run result was an artifact of the simulation methodology, not a genuine signal.

### 5.7 Calibration

Post-hoc linear calibration reliably removes systematic bias (mean signed error → 0) and reduces MAE from ~0.70 to ~0.50. However, it cannot create correlation that doesn't exist in the raw predictions. Calibration slopes near zero (e.g., N: a=0.002 in Experiment F) indicate the model's rank-ordering carries no useful information for that trait on that sample.

### 5.8 Per-Trait Architecture Comparison

| Architecture | O | C | E | A | N | Macro r | API calls/sample |
|---|---|---|---|---|---|---|---|
| All-at-once (strict_style) | 0.105 | 0.112 | 0.043 | 0.128 | 0.147 | 0.107 | 1 |
| 5-predictor (per-trait) | 0.041 | 0.154 | 0.155 | 0.094 | 0.114 | 0.112 | 5 |

Per-trait predictors improved E (0.04→0.16) by eliminating cross-trait contamination, but this was offset by degradation in O, A, N due to few-shot anchoring to extreme values. Net effect: negligible improvement at 5× cost.

| Method | Dataset | Macro Pearson r | Reference |
|--------|---------|----------------|-----------|
| LIWC + SVM regression | MyPersonality (Facebook) | 0.20–0.35 | Kosinski et al., 2013 |
| GPT-4 zero-shot | Essays (self-report) | 0.10–0.18 | Various 2024 |
| Fine-tuned BERT | PANDORA | 0.15–0.25 | Gjurković et al., 2021 |
| **Big5Loop Llama 3.3 70B zero-shot** | **PANDORA** | **0.07–0.12** | **This work** |
| Big5Loop Llama 3.3 70B zero-shot | PERSONAGE (NLG) | 0.457 | Phase 4 |

Our result is consistent with published zero-shot LLM performance on self-report personality prediction from single text samples. The gap with PERSONAGE (0.457 vs 0.10) reflects the GT–text alignment difference, not detector quality.

---

## 7. Implications for Big5Loop Production System

1. **Single-message detection is a weak signal, not a diagnosis.** The system should treat each prediction as one noisy data point.

2. **Temporal aggregation is essential.** In production, Big5Loop accumulates predictions across multiple user interactions. Averaging N predictions reduces noise by a factor of √N. After ~25 interactions, effective r could reach ~0.50.

3. **Onboarding questionnaire recommended.** Pairing LLM-based detection with a short BFI questionnaire (e.g., BFI-10) at onboarding provides a reliable baseline. LLM detection then tracks personality drift over time.

4. **Trait-specific reliability varies.** Agreeableness and Neuroticism are the most detectable from text; Extraversion is the least reliable and should be weighted accordingly in coaching decisions.

---

## 8. Limitations

- All experiments used a single LLM (Llama 3.3 70B). Multi-model ensembling was not tested.
- Calibration was fitted and evaluated on the same samples (no held-out calibration set). Reported calibrated r values may be slightly optimistic.
- PANDORA GT uses percentile-based BFI scores, not raw scores. The percentile→[-1,1] mapping is linear but may not preserve the original psychometric scale properties.
- The Pandora test set has 1 post per user — multi-post aggregation could not be tested empirically.

---

## 9. Conclusion

Zero-shot LLM personality detection from single Reddit posts achieves macro Pearson r ≈ 0.10 against self-report BFI ground truth, regardless of prompt engineering strategy. This ceiling is driven by the fundamental construct mismatch between observable writing style and self-reported personality, not by detector limitations — as demonstrated by the same model achieving r = 0.457 on PERSONAGE where GT–text alignment is high.

For the Big5Loop production system, this means single-message personality detection provides a weak but real signal that becomes actionable through temporal aggregation across multiple user interactions.

## Appendix B: PERSONAGE Dataset Provenance

The PERSONAGE dataset (Mairesse & Walker, 2007, 2008) contains **580 utterances generated by the PERSONAGE NLG system** — a parameterizable natural language generator that encodes Big Five personality traits through over 60 linguistic features (hedging frequency, verbosity, lexical choice, discourse markers, etc.). The texts are **not human-written**; they are machine-generated restaurant reviews/comparisons. Human judges then rated each utterance for perceived personality using the Ten-Item Personality Inventory (TIPI, Gosling et al., 2003) on a 1–7 Likert scale.

This design creates near-perfect alignment between ground truth and text content: the GT measures perceived personality of the text, and the text was engineered to express specific personality levels. This fundamentally differs from PANDORA, where GT measures the author's self-reported personality — a construct only loosely correlated with writing style.
