# Big5Loop Evaluation Data — Datasource Description

This document describes the data sources used for simulated evaluation of Big5Loop's personality-aware dialogue system.

---

## 1. Overview

| Datasource | Location | Rows | Purpose |
|------------|----------|------|---------|
| BIG5-CHAT (raw) | `raw/big5_chat_dataset.csv` | ~100,000 | Dialogue pairs with Big Five trait labels |
| BIG5-CHAT (processed) | `processed/big5_chat_eval.jsonl` | 1,000 (sampled) | Big5Loop-ready evaluation set |
| BFI2 | `raw/bfi2_dataset.csv` | 1,733 | Personality profiles (60 items) |
| PERSONAGE | `raw/predefinedParams.*`, `raw/randomParams.*` | 580 | Utterances with human personality ratings |
| NEO-PI-R | `raw/35840_*` | — | Validated NEO personality instrument |
| **PANDORA Big5** | `pandora/raw/pandora_big5_*.csv` | (HF) | Reddit text + Big Five labels — Phase 5 benchmark |
| **APP (subset)** | `pandora/raw/app_*.csv` | (HF) | Automated Personality Prediction — optional cross-check |

## 2. BIG5-CHAT (Primary Evaluation Datasource)

### 2.1 Provenance

- **Name**: BIG5-CHAT
- **Source**: Hugging Face `wenkai-li/big5_chat`
- **URL**: https://huggingface.co/datasets/wenkai-li/big5_chat
- **Citation**: Li et al. (2024), arXiv:2410.16491
- **Paper**: "BIG5-CHAT: Shaping LLM Personalities Through Training on Human-Grounded Data" (ACL 2025)

### 2.2 Description

BIG5-CHAT is a large-scale dataset of 100,000 dialogues designed to ground language models in how humans express Big Five personality traits in text. It combines:
- **PsychGenerator**: Facebook posts annotated with Big Five trait scores
- **SODA**: Diverse social interaction dialogues

Each dialogue pair has a target personality (one of five traits at low or high level). The assistant response is written to convey that trait.

### 2.3 Schema (Raw)

| Field | Type | Description |
|-------|------|-------------|
| `env_idx` | int | Environment / scenario index |
| `trait` | str | Big Five trait: `openness`, `conscientiousness`, `extraversion`, `agreeableness`, `neuroticism` |
| `level` | str | `low` or `high` |
| `train_instruction` | str | System prompt with personality instruction |
| `train_input` | str | User message (PersonX utterance) |
| `train_output` | str | Expected assistant response (PersonY, target personality) |
| `head`, `relation`, `tail` | str | Atomic relation from ATOMIC |
| `literal` | str | Literal relation description |
| `narrative` | str | Narrative context |
| `personx`, `persony` | str | Character names |
| `original_index` | int | Index in source corpus |

### 2.4 Schema (Processed)

| Field | Type | Description |
|-------|------|-------------|
| `input` | str | User message (from `train_input`) |
| `expected_output` | str | Ground-truth assistant response (from `train_output`) |
| `trait` | str | Big Five trait |
| `level` | str | `low` or `high` |
| `ground_truth` | str | `{trait}_{level}` (e.g. `openness_low`) |

### 2.5 Distribution

- **Traits**: 5 (openness, conscientiousness, extraversion, agreeableness, neuroticism)
- **Levels**: 2 (low, high)
- **Per cell**: 20,000 raw; configurable sample (e.g. 100) in processed output

### 2.6 Preprocessing

- Drop null `train_input` / `train_output`
- Remove duplicate `(input, trait, level)` rows
- Optional stratified sampling by `(trait, level)`
- Output: JSONL and CSV in `processed/`

## 3. BFI2

### 3.1 Provenance

- **Name**: Big Five Inventory 2 (BFI-2)
- **Source**: ShinyItemAnalysis R package (CRAN)
- **Reference**: Soto & John (2017), BFI-2

### 3.2 Description

Self-report personality data from 1,733 respondents. 60 items on 5-point Likert scale (1 = Disagree strongly, 5 = Agree strongly). Five domains with three facets each:

- **Extraversion (E)**: Sociability, Assertiveness, Energy Level
- **Agreeableness (A)**: Compassion, Respectfulness, Trust
- **Conscientiousness (C)**: Organization, Productiveness, Responsibility
- **Negative Emotionality (N)**: Anxiety, Depression, Emotional Volatility
- **Open-Mindedness (O)**: Intellectual Curiosity, Aesthetic Sensitivity, Creative Imagination

### 3.3 Schema

| Field | Type | Description |
|-------|------|-------------|
| `i1`–`i60` | int (1–5) | Item responses |
| `Gender` | int | 0 = female, 1 = male |
| `Age` | int | Age in years |
| `Educ` | str | Education level |

### 3.4 Use in Big5Loop

Derive target OCEAN profiles; simulate users with known personality; compare Big5Loop inferred OCEAN to ground truth.

## 4. PERSONAGE

### 4.1 Provenance

- **Name**: PERSONAGE
- **Source**: Mairesse & Walker (2006–2008), University of Sheffield
- **URL**: https://farm2.user.srcf.net/research/personage/README.html

### 4.2 Description

580 utterances (restaurant recommendations) generated by the PERSONAGE NLG system, annotated with human judge ratings (1–7) for Big Five traits. Used to evaluate style conveyed by personality parameters.

### 4.3 Schema (Tab Format)

| Field | Description |
|-------|-------------|
| `id` | Utterance ID (e.g. `extra_high-recommend-Pepolino-4`) |
| `avg.extra`, `avg.ems`, `avg.agree`, `avg.consc`, `avg.open` | Mean human ratings (1–7) |
| `avg.naturalness` | Naturalness rating |
| `realization` | Generated utterance text |

### 4.4 Use in Big5Loop

Compare Big5Loop output style to human personality ratings; test naturalness.

## 5. NEO-PI-R

### 5.1 Provenance

- **Name**: NEO Personality Inventory-Revised
- **Source**: D-Scholarship@Pitt, https://d-scholarship.pitt.edu/35840/
- **License**: CC BY-ND
- **Author**: Paul Pilkonis (2019)

### 5.2 Description

Validity study data for NEO-PI-R. SPSS (`.sav`) format with codebook. Alternative validated personality instrument for cross-checking profiles.

### 5.3 Files

- `35840_Validity_NEO-Participant_NEW_FINAL.sav` — Participant data
- `35840_Validity_NEO_codebook.docx` — Codebook

## 7. PANDORA (Phase 5 benchmark)

### 7.1 Provenance

- **Paper**: Gjurković, M., Karan, M., Vukojević, I., Šnajder, J., & Dalbelo Bašić, B. (2020). *PANDORA Talks: Personality and Demographics on Reddit*. ACL 2020. [ACL Anthology](https://aclanthology.org/2020.acl-main.614/)
- **Hugging Face — Big5 targets**: [`jingjietan/pandora-big5`](https://huggingface.co/datasets/jingjietan/pandora-big5)
- **Hugging Face — personality prediction subset**: [`Fatima0923/Automated-Personality-Prediction`](https://huggingface.co/datasets/Fatima0923/Automated-Personality-Prediction)

### 7.2 Description

Large Reddit-derived corpus with Big Five–related labels and demographics; suitable for **text → trait** benchmarking of Big5Loop’s detector against held-out ground truth.

### 7.3 Local layout

- Download: `python scripts/download_pandora.py` → `pandora/raw/*.csv`
- Protocol: `PHASE5-SPECIFICATION.md` (canonical); `PHASE5-PANDORA.md` (overview)
- Preprocess: map dataset-specific columns to `O, C, E, A, N` in `[-1, 1]` or document scaling vs Big5Loop outputs.
