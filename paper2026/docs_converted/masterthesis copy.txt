Adaptive LLM-Based Chatbot with Personality-Aware Dialogue for Human-Centered Applications

Project / Job Description (max.

characters)

This master’s thesis focuses on the development of a modular, data-driven chatbot system that
integrates psychological user modeling with state-of-the-art large language model (LLM)
architectures. The objective is to build a functional proof-of-concept chatbot capable of detecting
users' Big Five personality traits through natural language interaction and dynamically adapting its
responses using computational principles from the Zurich Model of Social Motivation.
The project places a strong emphasis on system architecture and applied machine learning. The
chatbot framework will be built using modern LLM development tools (e.g., LangChain or
Haystack), enabling context-aware, multi-turn dialogue management and seamless integration of
advanced components such as retrieval-augmented generation (RAG), prompt engineering, and
rule-based or model-driven adaptation logic.
From a data science perspective, the student will work on designing and evaluating algorithms for
real-time personality trait inference, multi-modal user state modeling, and behavioral response
generation. This includes developing and fine-tuning components for robust personality prediction
and consistent response modulation in extended conversations.
The project also includes a mandatory empirical evaluation via a small-scale user study involving
older adults. The study will assess the chatbot's interaction quality, emotional responsiveness, and
perceived personalization—providing valuable insights into real-world applicability and user
acceptance of psychologically adaptive dialogue systems.
The final deliverable will be a flexible and extensible chatbot prototype, accompanied by evaluation
results, demonstrating how psychological models can be computationally operationalized within
modern conversational AI systems for human-centered applications.
