# Expert Analysis Directory - APA Style Publication Figures

**Purpose:** Centralized repository for all statistical analysis, APA-style figure generation, and publication-quality data visualization.

**Status:** Production-Ready ✅  
**Last Updated:** February 1, 2026

## 📁 Quick Navigation

- `scripts/` - All Python analysis & plotting scripts (5 files)
- `data/` - All data files (raw + processed, 14 files)  
- `figures/` - All generated plots (30 PNG + PDF files)
- `notebooks/` - Jupyter notebooks (2 files)
- `docs/` - Complete documentation (13 guides)
- `config/` - Configuration files (requirements.txt)

---

##  Scripts That Generate APA-Style Figures

### 1. **enhanced_statistical_analysis.py**  
Generates Figures 06-11 (statistical analysis plots)

### 2. **visualization_config.py**  
APA publication style configuration module

### 3. **academic_data_quality_plots.py**  
CONSORT-style data quality figures

---

## 🚀 Quick Start

```bash
# Install dependencies
cd expert
pip install -r config/requirements.txt

# Run main analysis
cd notebooks
jupyter notebook statistical_analysis_enhanced.ipynb

# Or run scripts directly
cd scripts
python enhanced_statistical_analysis.py
```
