## ?? Core Scripts

### Main Analysis Scripts (USE THESE!)

| Script | Purpose | Run Time | Output |
|--------|---------|----------|--------|
| **visualization_config.py** | Unified styling configuration | N/A | Config object |
| **master_analysis.py** | Core statistical analysis | ~15s | 4 figures + 2 CSVs |
| **create_diagrams.py** | System architecture diagrams | ~5s | 2 figures |
| **personality_analysis.py** | Optional OCEAN analysis | ~5s | 1 figure + 1 CSV |

### Execution Order

```bash
# Required
python3 master_analysis.py      # Figures 01-04
python3 create_diagrams.py      # Figures 06-07

# Optional
python3 personality_analysis.py # Figure 05
```

### Script Details

#### visualization_config.py
```python
# Import in your scripts
from visualization_config import configure_matplotlib, save_figure

# Apply standards
configure_matplotlib()

# Create and save figures
fig, ax = create_plot()
save_figure(fig, 'my_figure', 'figures')
```

**Key Features:**
- Publication standards (300 DPI, colorblind-friendly)
- Figure templates (single, double, quad panels)
- Styling utilities
- Central configuration

#### master_analysis.py
```python
# Standalone execution
python3 master_analysis.py
```

**Analysis Pipeline:**
1. Load data (regulated + baseline)
2. Quality assessment
3. Convert to numeric scores
4. Calculate descriptive statistics
5. Calculate effect sizes
6. Generate 4 visualizations

**Outputs:**
- `01_performance_comparison.png`
- `02_effect_sizes.png`
- `03_personality_needs.png`
- `04_sample_quality.png`
- `analysis_results_descriptive.csv`
- `analysis_results_effect_sizes.csv`

#### create_diagrams.py
```python
# Standalone execution
python3 create_diagrams.py
```

**Creates:**
- `06_system_architecture.png` - System pipeline
- `07_study_workflow.png` - Research methodology

**Features:**
- A4-optimized dimensions
- Clean, professional styling
- High-resolution (300 DPI)

#### personality_analysis.py
```python
# Optional - for supplementary analysis
python3 personality_analysis.py
```

**Generates:**
- `05_personality_profiles.png` - OCEAN dimensions
- `analysis_results_personality_profiles.csv` - Profile summary

**Use When:**
- Need detailed personality distribution analysis
- Creating supplementary materials
- Exploring OCEAN patterns
