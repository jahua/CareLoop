2. STRESS LEVEL METRICS (LLM Evaluator) - Added to Section 4.6
   
   Stress Classification (0-4):
   ✓ Level 0: None/minimal (calm, organized, solutions-focused)
   ✓ Level 1: Mild (some concern, manageable tone)
   ✓ Level 2: Moderate (clear stress, but coping present)
   ✓ Level 3: High (strong emotion, limited coping resources)
   ✓ Level 4: Crisis (acute distress, danger signals)
   
   Detected Signals (LLM Analysis):
   ✓ Lexical markers (overwhelmed, exhausted, trapped, etc.)
   ✓ Emotional intensity (ALL CAPS, repeated punctuation)
   ✓ Problem escalation (new/worsening problems mentioned)
   ✓ Behavioral indicators (loss of coping language)
   
   Stress Drivers Identified:
   ✓ Caregiver burden (role strain, guilt)
   ✓ Benefit navigation complexity (confusion, frustration)
   ✓ Self-care neglect (exhaustion, isolation, sleep)
   ✓ Role strain (competing responsibilities)
   ✓ Financial stress (income loss concerns)
   ✓ Social isolation (lack of support)
   
   System Response Appropriateness:
   ✓ Level 0-1: Balanced tone, proactive guidance → Score 2
   ✓ Level 2: Validate, coping strategies → Score 2
   ✓ Level 3: Reassurance, break problems down → Score 2
   ✓ Level 4: Crisis protocol, professional referral → Score 2
